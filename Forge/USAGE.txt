-------------------------------------------
Source Installation Information for APCS
-------------------------------------------

    1.  Begin by running, from the parent directory, install-eclipse.cmd or install-intellij.cmd
        depending on which IDE you will be using.

    2a. [for Eclipse users] Open a new eclipse workspace, the directory is located in
        the "eclipse" folder of this current directory.

    2b. [for IntelliJ users] Find the file with the file extension .ipr, or open
        IntelliJ, browse to this directory, and find the file with the IntelliJ
        symbol.

    3.  To compile and run, no special instructions are required, just run as
        normal and the game should launch.
